  1. Export your résumé as a PDF.
  2. Name it exactly:  resume.pdf
  3. Drop it in this folder (assets/resume/resume.pdf), replacing nothing —
     it just needs to sit alongside this file.
  4. Delete this .txt file (optional, just tidiness).

The site already links to assets/resume/resume.pdf from the nav bar, the
hero "Download Résumé" button, and the Résumé section's embedded preview —
so once the file exists at that path, all three start working with no
other changes needed.
